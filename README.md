# 🧠 Learning NumPy (Structured Practice Repository)

## 🚀 Overview

This repository is my  **structured, hands-on journey to learning
NumPy** through implementation-focused scripts.

Instead of passive learning, I exlored every concept through
**code-first experimentation**, covering core NumPy operations step by
step.

------------------------------------------------------------------------

## 📂 Repository Structure

The files are organized in a logical progression:

1.  **MultiDimensional_Arrays.py**\
    → Understanding NumPy arrays and shapes

2.  **Slicing.py**\
    → Indexing and slicing techniques

3.  **Arithmetic.py**\
    → Element-wise operations

4.  **Broadcasting.py**\
    → Efficient operations across arrays

5.  **Aggregate_Functions.py**\
    → Sum, mean, min, max, etc.

6.  **Filtering.py**\
    → Boolean masking and conditional selection

7.  **RNG.py**\
    → Random number generation

------------------------------------------------------------------------

## 🧠 Learning Approach

-   Focused on **concept clarity through implementation**
-   Each file isolates a **single concept**
-   Progression is designed to move from **basics → intermediate usage**
-   Emphasis on understanding **how NumPy works internally**

------------------------------------------------------------------------

## ⚙️ How to Use

Run files in order:

``` bash
python MultiDimensional_Arrays.py
```

Follow the sequence listed above for best understanding.

------------------------------------------------------------------------

## 🔥 Key Takeaways

-   NumPy enables **vectorized operations** (faster than loops)
-   Broadcasting allows **efficient computation without explicit
    iteration**
-   Array operations form the foundation of:
    -   Machine Learning
    -   Data Science
    -   Scientific Computing
